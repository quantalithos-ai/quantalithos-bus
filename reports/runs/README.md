# Run Reports

Each fixed run id gets a dedicated report directory at `reports/runs/<run_id>`.

Expected run report files:
- `summary.md`
- `evidence-index.md`
- `gate-results.md`
- `coverage-matrix.md`
- `config-summary.md`
- `redaction-check.md`
- `artifact-index.md`
- `suites/<suite>.md`
- `evidence/EV-BUS-<AREA>.md`
