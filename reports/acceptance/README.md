# Acceptance Reports

Acceptance handoff material lives under `reports/acceptance`.

Expected files:
- `<run_id>-index.md`
- `handoff.md`
- `veto-checklist.md`
- `risk-acceptance.md`
- `open-issues.md`

These files may be generated as drafts by scripts, but they must be reviewed and
completed by a human or an agent before formal acceptance.
